cShare
============================

[cShare](https://plugin.caohongyuan.cn/share) - 超轻量级平台分享插件，简洁易用、开源免费。

### 使用方法：

```html
<!-- 引入插件js -->
<script src="https://plugin.caohongyuan.cn/cshare/cshare.js"></script>

<!-- 加载logo -->
<div id="cshare"></div>
```
更多示例请点击：https://plugin.caohongyuan.cn/share
